## Install steps (Mac/Linux)

Install uv:

~~~
$ curl -LsSf https://astral.sh/uv/install.sh | sh
~~~

*Comment* : This will install the latest version of Python to
`~/.local/bin` (with `~` a shell alias to the home folder).
If this folder is not in `PATH` (check with `echo $PATH`) 
you can enforce this in the shell startup file (.bashrc or .zshrc) by
redefining `PATH`

~~~
$ export PATH=~/.local/bin:$PATH
~~~
or with an alias
~~~
$ alias uv=~/.local/bin/uv
~~~

~~~
$ which uv
/home/bb1000-vt26/.local/bin/uv
~~~

Installing Python with `uv`:

~~~bash
$ uv python install
~~~

~~~bash
$ which python
~~~

is looking for programs matching commands that you type

## Command-line commands used (Mac-Linux)

* `curl`
    - get content from the web on the command line
    - the install line uses `curl` to download a script `install.sh` passed on as
      input to the Bourne shell interpreter `/bin/sh`
    - The pipe character `|` is used to pass on the output from one program as input to
      another
* `pwd`
    - print work directory:
    - within the file system, where am I?
* `ls`
    - list files:
    - what is in the work directory?
* `mkdir`
    - make directory 
    - create a new subfolder with the given name
* `cd`
    - change directory
    - if the argument is a folder that becomes the current work directory
    - without argument the home directory becomes current directory
      (mac/linux). In windows it does not change.
* `source`
    - execute the shell commands in a script file within the context of the current
      shell. 
    - the difference with `bash script-file` is that commands and definitions
      run in a subprocess and down not modify the current shell environment
